# CCNAv7: Enterprise Networking, Security, and Automation

## Certificate
Completed Cisco Networking Academy course: CCNAv7 – Enterprise Networking, Security, and Automation.

This course focuses on enterprise network design, advanced routing concepts, network security, and automation technologies used in modern networks.

---

## Key Topics Learned

### Enterprise Network Architecture
- Design and structure of enterprise networks
- Scalable network infrastructure
- Network redundancy and reliability

### Advanced Routing Concepts
- Dynamic routing protocols
- OSPF (Open Shortest Path First)
- Route selection and routing tables

### Network Security
- Securing network devices
- Access Control Lists (ACLs)
- Device hardening
- Protecting network infrastructure from attacks

### Network Automation
- Introduction to network automation
- Using APIs for network management
- Basic automation concepts in network operations

---

## Tools and Technologies

- Cisco Packet Tracer
- Cisco IOS CLI
- OSPF configuration
- Access Control Lists (ACL)

---

## Key Skills Developed

- Understanding enterprise network infrastructure
- Configuring and troubleshooting routing protocols
- Implementing basic network security practices
- Understanding network automation concepts
- Managing and monitoring network performance

---

## Relevance to Cybersecurity

Enterprise networking and security concepts are essential for cybersecurity professionals. Understanding routing, network segmentation, and device security helps security analysts monitor network activity, identify suspicious behavior, and protect organizational infrastructure.
