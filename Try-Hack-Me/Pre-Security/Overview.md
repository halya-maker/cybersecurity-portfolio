# TryHackMe – Pre Security Path

The Pre Security learning path on TryHackMe introduces the fundamental concepts required to begin a career in cybersecurity. This path covers networking, operating systems, web technologies, and basic security concepts.

---

Status: 70% Completed
Platform: TryHackMe
Learning Path: Pre Security

## Module 1: Introduction to Cyber Security

### Topics Covered
- Offensive Security
- Defensive Security
- Careers in Cybersecurity

### Key Takeaways
- Cybersecurity involves protecting systems, networks, and data from cyber threats.
- Offensive security focuses on finding vulnerabilities.
- Defensive security focuses on protecting and monitoring systems.

---

## Module 2: Network Fundamentals

### Topics Covered
- What is Networking
- Introduction to LAN
- OSI Model
- Packets and Frames
- Extending Your Network

### Key Takeaways
- Networks allow devices to communicate and share resources.
- The OSI model explains how data travels across networks.
- Understanding packets and frames helps analyze network traffic.

---

## Module 3: How The Web Works

### Topics Covered
- How Websites Work
- HTTP in Detail
- DNS in Detail
- Putting It All Together

### Key Takeaways
- DNS translates domain names into IP addresses.
- HTTP is the protocol used for communication between web servers and browsers.
- Understanding web communication is essential for detecting web-based attacks.

---

## Module 4: Computer Fundamentals

### Topics Covered
- Computer Types
- Inside a Computer System
- Client-Server Basics
- Virtualization Basics
- Cloud Computing Fundamentals

### Key Takeaways
- Computers consist of hardware components such as CPU, RAM, and storage.
- Client-server architecture is used in modern networks.
- Virtualization allows multiple systems to run on one physical machine.

---

## Module 5: Operating Systems Basics

### Topics Covered
- Windows Basics
- Windows CLI Basics
- Linux CLI Basics
- Operating System Security

### Key Takeaways
- Operating systems manage hardware and software resources.
- Linux command line is widely used in cybersecurity.
- Understanding system security helps detect unauthorized activities.

---

## Module 6: Software Basics

### Topics Covered
- Data Representation
- Data Encoding
- Python Basics
- JavaScript Basics
- Database SQL Basics

### Key Takeaways
- Data can be represented in binary, hexadecimal, and other formats.
- Basic scripting helps automate cybersecurity tasks.
- SQL is used to interact with databases.

---

## Module 7: Attacks and Defenses

### Topics Covered
- Cryptography Concepts
- CIA Triad
- Offensive Security Basics
- Defensive Security Basics

### Key Takeaways
- The CIA Triad (Confidentiality, Integrity, Availability) is the foundation of information security.
- Cryptography protects sensitive data.
- Understanding both attack and defense helps security professionals detect and respond to threats.
