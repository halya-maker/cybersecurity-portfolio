# 🔐 SOC Analyst Internship Experience

This repository showcases my hands-on experience as a **Security Analyst Intern** at Nexagate Sdn. Bhd., where I worked in a real-world **Security Operations Center (SOC)** environment.

---

## 📌 Overview

During my internship, I was involved in daily SOC operations including:

- SIEM monitoring using Splunk  
- Alert investigation and incident escalation  
- DNS monitoring and analysis  
- Web Application Firewall (WAF) monitoring (Imperva & Akamai)  
- Threat intelligence reporting (MITRE ATT&CK mapping)  

This experience provided practical exposure to real cybersecurity workflows and strengthened both my technical and analytical skills.

---

## 🧠 Key Skills Gained

- Log analysis & correlation  
- Alert validation (True Positive vs False Positive)  
- Incident handling & escalation  
- Threat analysis & reporting  
- DNS & Web security monitoring
